# LittleStyles
A little css minifier made in java ;)   

## Configuration file
```styleconfig
  input:      ./your/input/url.css
  output:     ./your/output/directory
  name:       master.css
  readable:   false
```

## Using LittleStyles
Use the latest binary with
```shell
  java -jar ./lit-v1.0.jar
```
